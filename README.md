# LTE - Rewrite in Progress

> [!CAUTION]
> **SKELETON STATE**
>
> This project is currently undergoing a complete rewrite. The codebase is actively being reconstructed and is currently in a **skeleton state**.
>
> - **Most features are missing or non-functional.**
> - **Builds may be unstable.**
> - **This is not ready for daily use.**

## About
LTE is a fully local-only Android client for YouTube. This rewrite aims to modernize the architecture and remove all dependencies on Piped instances, focusing purely on device-centric, offline-first functionality.
